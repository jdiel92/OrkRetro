# OrkRetro React (educacional)

Passos para rodar (Windows):
1. Abra terminal em:
   `c:\Users\Julian\Desktop\Estudo Tecnologia\Teste IA`

2. Instale dependências:
   `npm install`

3. Se você já criou o backend (server.js) rode-o em outra janela:
   `node server.js`  (por padrão roda em http://localhost:3000)

4. Rode frontend React (Vite):
   `npm run dev`
   Abra http://localhost:5173

Observações:
- O frontend tenta consumir `/api/*`. Se o backend não estiver rodando, há fallbacks locais mínimos.
- Código simples, comentado em português, nível trainee/Jr.
